How much RAM will I need?<\/h2>\n\n\n\n

In order to run Questbook, your system must have atleast 512MB <\/strong>or 1GB <\/strong>of system memory.<\/p>\n<\/div>\n\n\n\n

\n

How much storage do I need?<\/h2>\n\n\n\n

Your device must have 5GB <\/strong>of free storage or 10GB <\/strong>for more expansion.<\/p>\n<\/div>\n<\/div>\n\n\n\n

What is the recommended screen size?<\/h2>\n\n\n\n

The recommended screen size is 1024 x 768<\/strong> or higher. For devices with smaller screens will be 800 x 800 <\/strong>in the 4:3 aspect ratio. <\/p>\n<\/div>\n\n\n\n

\n

What CPU do I need?<\/h2>\n\n\n\n
